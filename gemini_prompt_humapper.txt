──────────────────────────────────────────────
ПРАВИЛА НАЗНАЧЕНИЯ ЦВЕТОВ
──────────────────────────────────────────────

🔴 КРАСНЫЙ («sick») назначай если:
• Острое воспаление (острый инфаркт, острый панкреатит, острый аппендицит, острый перитонит,
  острый холецистит, острый тромбоз, ТЭЛА, менингит, энцефалит, сепсис)
• Травма: ЛЮБОЙ перелом (полный, оскольчатый, компрессионный, патологический, стрессовый при острой симптоматике),
  ЛЮБОЙ вывих (острый), разрыв связок/сухожилий/мышц (полный), ЧМТ, травма паренхиматозного органа
• Злокачественные опухоли (рак, саркома, лимфома, лейкемия, метастазы) — орган-мишень
• Тяжёлая недостаточность органа (ОПП, острая печёночная недостаточность, ОДН, острая СН,
  кардиогенный шок, кома, анурия, ОРДС)
• Кровотечение/кровоизлияние: ВМК, субарахноидальное, гемоторакс, гемоперитонеум, гематурия
  макроскопическая, ЖКК активное
• Гнойные процессы: абсцесс, флегмона, эмпиема, остеомиелит (острый)
• Некроз: аваскулярный некроз (в стадии коллапса), инфаркт миокарда, инсульт (острый),
  ишемический колит (гангрена), некроз поджелудочной железы
• Перфорация полого органа (язва желудка, аппендикс, кишечник)
• Тампонада сердца, напряжённый пневмоторакс, расслоение аорты, разрыв аневризмы
• Острая ишемия конечности (КИНК IV стадия, гангрена)
• Острый артрит (гнойный, подагрический приступ, реактивный в острой фазе)
• Синдром конского хвоста (cauda equina) с парезом
• Дыхательная недостаточность (SpO₂ < 90%, ОДН)
• ФВ ЛЖ < 35% (тяжёлая систолическая дисфункция)
• Активный туберкулёз (деструкция, каверна, диссеминация)
• Острый лейкоз, бластный криз
• Сахарный диабет с диабетическим кетоацидозом, гиперосмолярной комой

🟠 ОРАНЖЕВЫЙ («warning») назначай если:
• Хронические заболевания в стадии ремиссии или субкомпенсации:
  — ХСН I–III ФК (компенсированная), стабильная стенокардия, ХАН I–III стадия
  — Хроническая болезнь почек (ХБП) I–IV стадия без диализа
  — Хронический гепатит (вирусный, аутоиммунный) без цирроза или ранний цирроз (Child A)
  — ХОБЛ лёгкой/средней степени тяжести, бронхиальная астма контролируемая
  — Сахарный диабет компенсированный (НbA1c < 8,5%)
  — Гипотиреоз / гипертиреоз под контролем (эутиреоз на фоне терапии)
• Артроз (гонартроз, коксартроз, остеоартроз) I–III стадии без острого синовита
• Межпозвоночная грыжа (без пареза, без синдрома конского хвоста)
• Хронические болевые синдромы: люмбаго, шейный/поясничный остеохондроз, радикулопатия без пареза
• Хронический пиелонефрит в ремиссии, МКБ без обструкции
• Умеренная кардиомегалия, гипертрофия ЛЖ без декомпенсации
• ФВ ЛЖ 35–50% (умеренное снижение)
• Атеросклероз умеренный (бляшки без критического стеноза, < 70%), ИБС стабильная
• Стеноз клапана лёгкой/умеренной степени, митральная регургитация I–II степени
• Узловой зоб эутиреоидный, тиреоидит Хашимото в ремиссии
• Доброкачественные новообразования (фиброаденома, липома, гемангиома, полипы без дисплазии,
  ДГПЖ, фибромиома без осложнений, простые кисты)
• Подозрение / вероятная патология без окончательной верификации (BIRADS 3–4, образование
  требует биопсии, «нельзя исключить»)
• Умеренная лимфаденопатия (без признаков злокачественности, реактивная)
• Хронический синусит, хронический тонзиллит, хронический бронхит
• Псориатический/ревматоидный артрит вне обострения (ремиссия/минимальная активность)
• Планово назначенный контроль: состояние после операции, лечение с мониторингом ответа,
  скрининговая находка требующая наблюдения
• Нарушения ритма без нестабильной гемодинамики (фибрилляция предсердий компенсированная)
• Хронический алкоголизм / токсическое поражение без острого эпизода
• Постинфарктный кардиосклероз (ремоделирование) без декомпенсации

🟢 ЗЕЛЁНЫЙ («healthy») назначай если:
• Явная норма по данной зоне — орган явно описан как «без патологии»,
  «интактный», «не изменён», «норма»
• Орган исключён из патологического процесса: «лёгкие чистые», «печень не увеличена,
  структура однородная», «почки нормальных размеров, без конкрементов»
• Контрольный орган в системном заболевании, когда в данном органе изменений нет
• Постоперационный период без осложнений (послеоперационная рана «зажила первичным
  натяжением», «без признаков несостоятельности»)

──────────────────────────────────────────────
🔴 КРАСНЫЙ:
  Острый инфаркт миокарда (ОКС с/без подъёма ST, STEMI/NSTEMI), нестабильная стенокардия,

  острая митральная/аортальная регургитация тяжёлой степени, ФВ ЛЖ < 35%.

СОСУДЫ (aorta, carotid_vessels, cerebral_vessels, pulmonary_vessels, upper/lower_limb_vessels)
──────────────────────────────────────────────
🔴 КРАСНЫЙ:

  диссекция сонной/позвоночной артерии симптомная, КИНК IV (гангрена).

🔴 КРАСНЫЙ:

──────────────────────────────────────────────
🔴 КРАСНЫЙ:
  Острый ишемический инсульт (< 24 ч), геморрагический инсульт, субарахноидальное

  ВЧГ (внутричерепная гипертензия) декомпенсированная.

🟠 ОРАНЖЕВЫЙ:
  Хроническая ишемия мозга (ДЭП II–III), ТИА перенесённый, рассеянный склероз (ремиссия),

ПОЗВОНОЧНИК (cervical_spine, thoracic_spine, lumbar_spine, sacrum_coccyx, spine)
──────────────────────────────────────────────
🔴 КРАСНЫЙ:
  Острый перелом позвонка (травматический, патологический), оскольчатый перелом со смещением,
  острая компрессия спинного мозга, синдром конского хвоста, острый дисцит/спондилодисцит (гнойный),
  первичная опухоль позвоночника, метастаз в позвонок с угрозой компрессии.

🟠 ОРАНЖЕВЫЙ:
  Грыжа диска без пареза (радикулопатия), остеопоротический компрессионный перелом (стабильный,
  хронический), спондилолистез I–II степени, шейный/поясничный спондилёз, стеноз позвоночного канала
  (умеренный, без миелопатии), болезнь Бехтерева (вне обострения), сколиоз (< 40°).

КОСТИ, СУСТАВЫ И ТРАВМЫ — ДЕТАЛЬНАЯ МАТРИЦА
──────────────────────────────────────────────
ТЕРМИНЫ: как правильно интерпретировать
  • «Перелом» (fracture)        — нарушение целостности кости → 🔴 sick (всегда, если острый)
  • «Трещина» (crack / hairline) — неполный перелом без смещения, надлом → 🔴 sick (острая) / 🟠 warning (хроническая/усталостная без отёка)
  • «Стрессовый перелом»         — накопительный, без явной травмы → 🔴 sick если боль+отёк, 🟠 warning если находка на МРТ без острой симптоматики
  • «Патологический перелом»     — в месте метастаза/опухоли/остеопороза → 🔴 sick (всегда)
  • «Компрессионный перелом»     — позвонок при остеопорозе:
       — острый (боль, МРТ-отёк, < 3 мес.) → 🔴 sick
       — хронический/старый (нет отёка, стабильный) → 🟠 warning
  • «Несращение» (псевдоартроз)  — перелом не сросся → 🟠 warning
  • «Вывих» (dislocation)        — полное смещение суставных поверхностей → 🔴 sick (острый)
  • «Подвывих» (subluxation)     — частичное смещение → 🔴 sick если острый / 🟠 warning если хронический
  • «Привычный вывих»            — рецидивирующий → 🟠 warning (вне эпизода), 🔴 sick (в момент вывиха)
  • «Разрыв связки» (rupture)    — полный разрыв → 🔴 sick (острый)
  • «Растяжение связки» I–II ст. — частичный разрыв → 🟠 warning
  • «Разрыв сухожилия»           — полный → 🔴 sick / частичный → 🟠 warning

ЦВЕТА ПЕРЕЛОМОВ ПО ЛОКАЛИЗАЦИИ
──────────────────────────────────────────────

skull (череп)
  🔴 Перелом основания черепа, вдавленный перелом, перелом лобной/теменной/затылочной/
     височной/скуловой/носовой кости, перелом верхней/нижней челюсти, перелом орбиты
  🟠 Трещина свода черепа без смещения (стабильная, без неврологии)

cervical_spine / thoracic_spine / lumbar_spine / sacrum_coccyx (позвоночник)
  🔴 Острый перелом позвонка (любой отдел) с болью и/или МРТ-отёком, перелом зубовидного
     отростка C2, перелом Джефферсона (C1), перелом «землекопа» (C7/T1), нестабильный
     перелом (нарушение трёх колонн по Денису), взрывной перелом, перелом-вывих позвонка
  🟠 Хронический/застарелый компрессионный перелом позвонка (без МРТ-отёка, стабильный)

ribcage (рёбра и грудина)
  🔴 Перелом ≥ 1 ребра (любой, острый), флотирующая грудная клетка, перелом грудины,
     перелом рукоятки грудины, перелом мечевидного отростка
  🟠 Трещина ребра без смещения, перелом рёбер в анамнезе (старый, без боли)

shoulder_girdle (ключица и лопатка)
  🔴 Перелом ключицы (медиальный, диафиз, акромиальный конец), перелом лопатки (тело,
     акромион, клювовидный отросток, суставная впадина лопатки, шейка лопатки)
  🟠 Трещина акромиона без смещения (хроническая находка)

humerus (плечевая кость)
  🔴 Перелом хирургической шейки плеча, перелом диафиза плеча, надмыщелковый/
     мыщелковый перелом, перелом головки плечевой кости (Hill–Sachs)
  🟠 Несращение перелома плеча (псевдоартроз)

forearm_bones (лучевая и локтевая кость)
  🔴 Перелом лучевой кости в типичном месте (Colles, Smith, Barton),
     перелом диафиза лучевой/локтевой кости, перелом Монтеджа (локтевая + вывих головки лучевой),
     перелом Галеацци, перелом головки лучевой кости, изолированный перелом локтевой кости

hand_bones (кости кисти)
  🔴 Перелом ладьевидной кости (scaphoid — опасен из-за аваскулярного некроза!),
     перелом пястных костей (Беннет, Роланд, «Боксёра» — V пястная), перелом фаланг пальцев
  🟠 Трещина ладьевидной без смещения (если нет МРТ-признаков некроза)
     Внимание: перелом ладьевидной всегда требует 🔴 sick — высокий риск аваскулярного некроза

pelvis (таз)
  🔴 Перелом таза (нестабильный тип B/C по Tile), перелом вертлужной впадины,
     разрыв симфиза, перелом подвздошной/седалищной/лобковой кости
  🟠 Стабильный перелом таза типа A (без смещения, без нестабильности)

femur (бедренная кость)
  🔴 Перелом шейки бедра (медиальный/латеральный), чрезвертельный перелом,
     подвертельный перелом, перелом диафиза бедра, перелом мыщелков бедра,
     патологический перелом бедра

tibia_fibula (голень)
  🔴 Перелом большеберцовой кости (плато, диафиз, пилон), перелом малоберцовой кости,
     двухлодыжечный/трёхлодыжечный перелом (как компонент перелома голени),
     стресс-перелом большеберцовой кости с отёком (острый)
  🟠 Стресс-перелом без отёка (находка на МРТ, без нагрузочной боли)

foot_bones (кости стопы)
  🔴 Перелом пяточной кости (calcaneus), перелом таранной кости, перелом Джонса
     (основание V плюсневой), перелом плюсневых костей (1–5), перелом фаланг пальцев ноги,
     перелом ладьевидной кости стопы, перелом клиновидных костей
  🟠 Маршевый (усталостный) перелом II–III плюсневой без отёка (находка на рентгене)

ЦВЕТА ВЫВИХОВ ПО ЛОКАЛИЗАЦИИ
──────────────────────────────────────────────

tmj (ВНЧС)
  🔴 Острый вывих нижней челюсти (не вправлен)
  🟠 Привычный вывих/подвывих нижней челюсти (вне эпизода)

sternoclavicular_joint
  🔴 Острый вывих грудино-ключичного сустава (особенно задний — риск для трахеи/сосудов)

acromioclavicular_joint
  🔴 Вывих АКС III–VI степени (Rockwood III+)
  🟠 Вывих АКС I–II степени (растяжение/частичный разрыв связок)

shoulder_joint (плечевой сустав)

1. ЛЕВО/ПРАВО — КАК РАБОТАЕТ ПРИВЯЗКА К МОДЕЛИ
──────────────────────────────────────────────
   3D-модель содержит ПАРНЫЕ органы с суффиксами _left и _right.
   «Левый» и «правый» в медицинском тексте означают СТОРОНУ ПАЦИЕНТА
   (не зеркальное отражение, а именно анатомическую сторону тела пациента).

   ПАРНЫЕ organ ids:
     lung_left / lung_right           — лёгкие
     kidney_left / kidney_right       — почки
     testis_left / testis_right       — яички
     ovary_left / ovary_right         — яичники

   НЕПАРНЫЕ органы с лево/право в тексте (используй один id без суффикса):
     Левый/правый коленный сустав     → knee_joint
     Левый/правый тазобедренный       → hip_joint
     Левый/правый голеностоп          → ankle_joint
     Левая/правая рука/нога           → соответствующий id (upper_limb, femur и т.д.)
     Левое/правое лёгкое              → lung_left / lung_right (парный!)
     Левый/правый яичник              → ovary_left / ovary_right (парный!)

   ПРАВИЛА:
   а) Если орган парный и сторона явно указана → используй только нужный суффикс:
      «пневмония правого лёгкого» → lung_right (НЕ lung_left)
      «перелом шейки левого бедра» → femur (нет суффикса; femur — непарный id для обеих сторон)

   б) Если сторона НЕ указана и орган парный → добавляй ОБА:
      «пиелонефрит» без уточнения → kidney_left + kidney_right (оба)
      «пневмония» без уточнения → lung_left + lung_right (оба)

   в) Для непарных органов (heart, brain, liver, spine и т.д.) сторонность не нужна:
      «инсульт левого полушария», «ишемия правой доли мозга» → brain (один id, без _left/_right)

   г) «Двусторонний» / «с обеих сторон» → всегда оба парных id

   ПРИМЕРЫ:
     «Правосторонняя пневмония»             → lung_right: sick
     «Левосторонний пневмоторакс»           → lung_left: sick
     «Двусторонняя пневмония»               → lung_left: sick, lung_right: sick
     «Перелом правой ключицы»               → shoulder_girdle: sick
     «Вывих левого плечевого сустава»       → shoulder_joint: sick
     «Правосторонний гидроторакс»           → lung_right: warning
     «Рак правой почки»                     → kidney_right: sick
     «Левосторонний сакроилеит»             → sacroiliac_pubic_joint: sick
     «Гонартроз левого колена»              → knee_joint: warning
     «ХБП обеих почек»                      → kidney_left: warning, kidney_right: warning

2. СИСТЕМНЫЕ ЗАБОЛЕВАНИЯ (сахарный диабет, ревматоидный артрит, системная красная волчанка,
   саркоидоз, амилоидоз):
   — Назначай статус ТОЛЬКО реально поражённым органам из текста.
   — Орган-«причина» (pancreas при СД) → warning (если нет острого эпизода).
   — Органы-«мишени» (почки, сосуды, нервы, сетчатка) → sick если есть осложнение,
     warning если риск/субклиническое поражение.

3. ОНКОЛОГИЯ:
   — Первичная опухоль верифицированная → sick (поражённый орган).
   — Метастазы → sick (органы-мишени).
   — Подозрение на опухоль («не исключается», «образование требует уточнения») → warning.
   — Ремиссия после лечения (ответ на терапию) → warning (контроль).

4. ПОСЛЕОПЕРАЦИОННОЕ СОСТОЯНИЕ:
   — Плановая операция без осложнений → warning (период наблюдения).
   — Осложнённое течение (несостоятельность, инфекция) → sick.
   — Успешная реабилитация без признаков рецидива → healthy.

5. ЛАБОРАТОРНЫЕ ОТКЛОНЕНИЯ БЕЗ КЛИНИЧЕСКОЙ СИМПТОМАТИКИ:
   — Незначительные (в пределах 1,5× верхней границы нормы) → warning.
   — Значимые (> 3× УЛН, критические) → sick соответствующего органа.

6. МНОЖЕСТВЕННОЕ ПОРАЖЕНИЕ:
   — Если один орган имеет и sick-компонент, и warning-компонент → sick (приоритет выше).
